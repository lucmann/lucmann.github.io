mesaдёӯзҡ„winsysеұӮ

Posted on 2021-09-12 In graphics Views: Disqus:

winsysжңүд»Җд№Ҳз”Ё

winsysеғҸдёҖдёӘжЎҘжўҒпјҢе®ғиҰҒе°ҶGPUжёІжҹ“зҡ„з»“жһңдј иҫ“еҲ°зӘ—еҸЈзі»з»ҹзҡ„framebufferпјҢз”ұжҳҫзӨәзі»з»ҹе°Ҷе…¶е‘ҲзҺ°еңЁеұҸе№•дёҠгҖӮ

mesaдёӯзҡ„е®һзҺ°

mesaдёӯwinsysжңүдёӨз§Қе®һзҺ°: дёҖз§ҚжҳҜеҹәдәҺdriзҡ„пјҢеҸҰдёҖз§ҚжҳҜдёҚеҹәдәҺdriзҡ„гҖӮдёӨиҖ…зҡ„дё»иҰҒеҢәеҲ«еңЁд»ҘдёӢеҮ зӮ№:

  • winsysжҸҗдҫӣзҡ„FBO(дёҠеӣҫдёӯзҡ„framebuffer)зҡ„жқҘжәҗ

еҹәдәҺdri

  • amdgpu_winsys
  • radeon_winsys
  • nouveau_winsys

еҹәдәҺdriзҡ„winsysзҡ„е®һзҺ°д»Јз ҒеңЁsrc/gallium/frontends/driзӣ®еҪ•

еҹәдәҺdri winsysзҡ„gallium driverзҡ„еҠ иҪҪ

жүҖжңүдҪҝз”ЁеҹәдәҺdriзҡ„winsysзҡ„gallium driversйғҪдјҡеҲ©з”ЁдёӢйқўзҡ„е®ҸеЈ°жҳҺе’Ңе®ҡд№үдёҖдёӘеҮҪж•°з¬ҰеҸ·(йҷӨдәҶswrast)

дҫӢеҰӮпјҢconst __DRIextension **__driDriverGetExtensions_r600(void);. дҪ еҸҜд»ҘеңЁзӣёеә”зҡ„еҠЁжҖҒеә“йҮҢжҹҘжүҫеҲ°иҜҘз¬ҰеҸ·гҖӮ
дёҠйқўе®Ҹдёӯзҡ„galliumdrm_driver_apiе’Ңgalliumdrm_driver_extensionsйғҪжҳҜе…ЁеұҖзҡ„гҖӮ

  • galliumdrm_driver_api
  • galliumdrm_driver_extensions

иҝҷйҮҢеҮәзҺ°зҡ„4дёӘж•°з»„е…ғзҙ йғҪжҳҜ__DRIextensionзҡ„еӯҗзұ»пјҢеҗҢж—¶д»–д»¬йғҪжҳҜе…ЁеұҖзҡ„гҖӮдҫӢеҰӮdriImageDriverExtensionиў«е®ҡд№үеңЁsrc/mesa/drivers/dri/common/dri_util.cж–Үд»¶дёӯ:

йҷӨдәҶиҝҷдёӘdriImageDriverExtensionпјҢеҹәдәҺdriзҡ„winsysиҝҳе®ҡд№үдәҶдёҖдёӘдёҺ__DRIimageжңүе…ізҡ„__DRIextension:

иҝҷдёӘжүҖи°“зҡ„image loader extensionжҳҜеңЁеӨҡGPUзҡ„еңәжҷҜдёӢдјҡз”ЁеҲ°пјҢд№ҹе°ұжҳҜеҪ“зі»з»ҹдёӯеҗҢж—¶еӯҳеңЁдёӨдёӘGPUж—¶пјҢеҸҜд»ҘйҖҡиҝҮи®ҫзҪ®mesaжҸҗдҫӣзҡ„зҺҜеўғеҸҳйҮҸDRI_PRIMEпјҢеҗҜз”ЁPRIMEжЁЎејҸпјҢжӯӨж—¶е…¶дёӯдёҖдёӘGPUдҪңдёәDisplay GPU(server GPU), з”ЁеҒҡX11жҳҫзӨәпјҢеҸҰеӨ–дёҖдёӘGPUдҪңдёәRender GPU, з”ЁеҒҡ3DжёІжҹ“пјҢеҪ“жҲ‘д»¬з”іиҜ·__DRI_IMAGE_BUFFER_FRONTж—¶пјҢжҲ‘д»¬дёҚиғҪзӣҙжҺҘдҪҝз”Ёpixmap,
еӣ дёәжүҖжңүpixmapsйғҪжҳҜиў«server GPUжүҖжңүпјҢиҝҷз§Қpixmapзҡ„ж јејҸеҸҜиғҪдёҚиғҪиў«render GPUжүҖзҗҶи§ЈгҖӮиҝҷз§Қжғ…еҶөдёӢпјҢеңЁloader_dri3_get_buffersдёӯпјҢдјҡеңЁrender GPUзҡ„VRAMйҮҢеҲӣе»әдёҖдёӘFake Front Buffer, жңҖеҗҺdri3_flush_front_bufferдјҡе°ҶFake Front Bufferзҡ„еҶ…е®№еҗҢжӯҘеҲ°зңҹжӯЈзҡ„Front BufferпјҢеҚіdisplay GPUзҡ„pixmapйҮҢгҖӮ

  • xcb_dri3_pixmap_from_buffer

иҝҷдёӘеҮҪж•°жҳҜз”ұc_client.pyиҮӘеҠЁз”ҹжҲҗзҡ„гҖӮ

иҖҢX Serverж”¶еҲ°иҝҷдёӘrequestеҗҺзҡ„еӨ„зҗҶеҮҪж•°еҰӮдёӢ:

еҸҜд»ҘзңӢеҲ°жңҖз»ҲжҳҜи°ғз”ЁдәҶpixmap_from_fdsжҲ–pixmap_from_fd, жҺҘдёӢжқҘзҡ„е·ҘдҪңз”ұdisplay GPUзҡ„kmdе®ҢжҲҗпјҢжңҖеҗҺеҪ“display GPU gbm_bo_importи°ғз”ЁдәҶdrmPrimeFDToHandleеҮҪж•°еҗҺпјҢX ServerиҝӣзЁӢиҝӣе…ҘеҶ…ж ёжҖҒпјҢжҺҘдёӢжқҘеҶ…ж ёDRMеӯҗзі»з»ҹзҡ„ioctlдјҡи°ғз”Ёdrm_prime_fd_to_handle_ioctl, е®ғдјҡи°ғз”Ёdisplay GPUзҡ„prime_fd_to_handle callback. ж•ҙдёӘиҝҮзЁӢе…¶е®һжҳҜе°Ҷжҳҫеӯҳ(buffer, зЎ®еҲҮиҜҙжҳҜback buffer)жҠҪиұЎжҲҗdma-buf(dma-bufе®һйҷ…дёҠжҳҜдёҖдёӘж–Үд»¶пјҢжүҖд»Ҙе®ғжңүfd, еҸҜдҫӣеңЁиҝӣзЁӢй—ҙдј йҖ’)жқҘе®һзҺ°з”ЁжҲ·еә”з”ЁиҝӣзЁӢдёҺX ServerиҝӣзЁӢй—ҙзҡ„Bufferе…ұдә«гҖӮ

дёӢйқўжҳҜеҶ…ж ёеҮҪж•°dma_buf_getзҡ„е®һзҺ°пјҢд»ҺиҝҷдёӘе°ҸеҮҪж•°зҡ„е®һзҺ°жҲ‘д»¬еҸҜд»Ҙжё…жҘҡең°зңӢеҲ°fd -> file -> dma_bufзҡ„иҪ¬жҚў

еҹәдәҺdri winsysзҡ„gallium driverзҡ„winsysе®һзҺ°

winsysжҳҜдёҖдёӘжЎҘжўҒпјҢе®ғдё»иҰҒиҰҒе®һзҺ°зҡ„е°ұжҳҜе°Ҷcolor bufferдј иҫ“еҲ°зӘ—еҸЈзі»з»ҹзҡ„framebuffer.йӮЈд№ҲеҹәдәҺdriзҡ„winsysжҳҜжҖҺд№Ҳе®һзҺ°иҝҷдёӘжЎҘжўҒзҡ„? е®ғдё»иҰҒдҫқиө–дёӢйқўдёӨдёӘеҜ№иұЎ:

  • DRIimage
    • __DRIimageRec
    • __DRIimageExtensionRec

DRIimageйҖҡиҝҮи°ғз”Ёpipe_screen.resource_createеҮҪж•°еҲӣе»әпјҢеҲӣе»әеҗҺиҝҳйңҖиҰҒqueryImageдёҖдәӣдҝЎжҒҜпјҢиҝҷдәӣдҝЎжҒҜдёәX11жүҖйңҖиҰҒгҖӮ

  • PIPE_RESOURCE_PARAM_NPLANES
  • PIPE_RESOURCE_PARAM_STRIDE
  • PIPE_RESOURCE_PARAM_OFFSET
  • PIPE_RESOURCE_PARAM_MODIFIER
  • PIPE_RESOURCE_PARAM_HANDLE_TYPE_FD

PIPE_RESOURCE_PARAM_HANDLE_TYPE_FDйҖҡиҝҮеҗ‘kernelеҸ‘йҖҒдёӢйқўиҜ·жұӮпјҢз”ұдёҖдёӘgem handleиҝ”еӣһдёҖдёӘfile descriptor. жңүдәҶиҝҷдёӘFD, дёҚеҗҢзҡ„иҝӣзЁӢй—ҙе°ұеҸҜд»Ҙе…ұдә«еҶ…еӯҳгҖӮжҜ”ж–№пјҢеҪ“xserverиҺ·еҸ–еҲ°иҝҷдёӘFDеҗҺпјҢе®ғеҸҜд»ҘйҖҡиҝҮDRM_IOCTL_PRIME_FD_TO_HANDLEеҸҲйҮҚж–°иҺ·еҸ–еҲ°gem boзҡ„handle.

  • X11 present extension
